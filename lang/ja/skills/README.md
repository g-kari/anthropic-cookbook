# Claudeスキル

Anthropic クックブックのスキルセクションへようこそ！このディレクトリには、Claudeが優れている特定のスキルと機能を紹介するガイドのコレクションが含まれています。各ガイドは特定のスキルの詳細な探求を提供し、潜在的な使用例、結果を最適化するためのプロンプトエンジニアリング技術、およびClaudeのパフォーマンスを評価するアプローチについて説明しています。

## ガイド

- **[Claudeによる分類](./classification/guide.ipynb)**: 複雑なビジネスルールと限られた訓練データのシナリオで、Claudeがどのように分類タスクを革新できるかを発見してください。このガイドでは、データ準備、検索拡張生成（RAG）を使ったプロンプトエンジニアリング、テスト、評価について説明します。

- **[Claudeによる検索拡張生成](./retrieval_augmented_generation/guide.ipynb)**: RAGを使ってドメイン固有の知識でClaudeの能力を向上させる方法を学びます。このガイドでは、RAGシステムをゼロから構築し、そのパフォーマンスを最適化し、評価スイートを作成する方法を実演します。要約インデックス化や再ランキングなどの技術が、質問応答タスクにおいて精度、再現率、全体的な正確性を大幅に改善する方法を学びます。

- **[文脈埋め込みによる検索拡張生成](./contextual-embeddings/guide.ipynb)**: RAGシステムのパフォーマンスを向上させる新しい技術の使用方法を学びます。従来のRAGでは、文書は効率的な検索のために通常小さなチャンクに分割されます。このアプローチは多くのアプリケーションでうまく機能しますが、個々のチャンクに十分なコンテキストが不足している場合に問題が生じる可能性があります。文脈埋め込みは、埋め込み前に各チャンクに関連するコンテキストを追加することでこの問題を解決します。セマンティック検索、BM25検索、再ランキングで文脈埋め込みを使用してパフォーマンスを向上させる方法を学びます。

- **[Claudeによる要約](./summarization/guide.ipynb)**: 複数のソースからの情報を要約し統合するClaudeの能力を探求します。このガイドでは、マルチショット、ドメインベース、チャンク化手法を含む様々な要約技術、および長文コンテンツと複数の文書を処理する戦略について説明します。また、芸術、主観性、適切なアプローチのバランスを取ることができる要約の評価についても探求します！

- **[Claudeによるテキストから SQL](./text_to_sql/guide.ipynb)**: このガイドでは、プロンプト技術、自己改善、RAGを使用して自然言語から複雑なSQLクエリを生成する方法について説明します。また、構文、データの正確性、行数などをテストする評価を使って、生成されたSQLクエリの正確性を評価し改善する方法についても探求します。

## はじめに

スキルガイドを始めるには、希望するガイドのディレクトリに移動し、`guide.ipynb`ファイルに記載されている指示に従ってください。各ガイドは自己完結型で、例や実験を再現するために必要なすべてのコード、データ、評価スクリプトが含まれています。
